# E-Commerce Sales Analytics Dashboard

## Overview

This project is an interactive E-Commerce Sales Analytics Dashboard built in Microsoft Excel to analyze sales performance, profitability, and business growth trends. The dashboard transforms raw sales data into actionable insights using KPI tracking, data visualization, and interactive filtering.

## Objectives

* Analyze overall sales and profit performance.
* Compare year-over-year sales growth.
* Identify top-performing regions and product categories.
* Detect loss-making products and sub-categories.
* Provide business insights for data-driven decision making.

## Features

* Interactive Slicers for dynamic filtering.
* KPI Cards for Sales, Profit, Loss, and Growth Metrics.
* Monthly Sales Trend Analysis with Moving Average.
* State-wise Sales Performance Analysis.
* Product Category and Sub-Category Analysis.
* Profitability and Loss Analysis.
* Business Insights and Key Findings.

## Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Moving Average Analysis
* Dashboard Design
* Data Visualization

## Key Insights

* Identified high-performing states contributing significantly to revenue.
* Analyzed year-over-year sales growth trends.
* Highlighted loss-making product categories affecting profitability.
* Evaluated product performance and regional sales distribution.
